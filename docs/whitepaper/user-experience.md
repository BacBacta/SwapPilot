## Settings

### Slippage Tolerance
- Preset options: 0.5%, 1%, 2%, 4%
- Custom input available
- Warnings for extreme values

### Transaction Speed
- Standard: Lower gas, slower confirmation
- Fast: Higher gas, faster confirmation
- Instant: Maximum gas, immediate confirmation
